
# ROpenDataSMM

`ROpenDataSMM`是一個方便R使用[OpenDataSMM](https://github.com/smmtw/OpenDataSMM)的介面，讓開發者能夠直接使用R讀取資料，而不需要手動下載。

## 安裝

透過GitHub:

``` r
# install.packages("devtools")
devtools::install_github("smmtw/ROpenDataSMM")
```

## 使用

列出目錄下所有檔案:

``` r
library('ROpenDataSMM')
files <- listFiles(path = "data")
files
#>           path type     size
#> 1 data/103.csv file 79625180
#> 2 data/104.csv file 80736469
#> 3 data/105.csv file 80080938
#> 4 data/106.csv file 82358357
#> 5 data/107.csv file 84463941
#> 6 data/108.csv file 68947535
```

使用關鍵字搜尋檔案:

``` r
files <- searchFiles(keyword = "07", path = "data")
files
#>           path type     size
#> 5 data/107.csv file 84463941
```
